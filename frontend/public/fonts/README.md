Place licensed Kamerik files in this folder so `globals.css` can load them.

Required filenames:

- `kamerik-105-cyrillic-light.woff2` (300)
- `kamerik-105-cyrillic-regular.woff2` (400-500)
- `kamerik-105-cyrillic-bold.woff2` (700)
- `kamerik-105-cyrillic-black.woff2` (800)

These map to `@font-face` entries in `src/app/globals.css`.
